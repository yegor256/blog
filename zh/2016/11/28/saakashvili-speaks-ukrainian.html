    Зачем Саакашвили говорит по украински?

    • Егор Бугаенко
    • комментарии
    0 0 0 0 0
    You can also read this article in English and 中文, translated by ChatGPT

    我对米哈伊尔·萨卡什维利抱有很大的好感，认为他是一个强大的人物，也是我认为唯一有很大未来的乌克兰政治家。然而，我完全不理解这种针对乌克兰语言的民粹主义。你为什么要这样做，米哈伊尔？

    我理解为什么阿扎罗夫、亚努科维奇以及其他亚采努克这样做。他们维持权力的主要手段始终是民粹主义： “一种针对人民大众需求而非精英需求的修辞风格”。

    乌克兰语是一种很好的方式来展示对人民群众需求的关心，特别是在目前与俄罗斯的紧张关系时。强调乌克兰语的特殊价值，任何政治家都很容易玩弄”外部敌人”的牌，将我们真正的乌克兰人和爱国者与他们这些不忠诚的帝国侵略者对立起来。语言就像一面旗帜，可以迅速升起：我说乌克兰语，那么我就是支持你们反对他们。同时，与真正的敌人合作或实际上成为他们。

    在战争期间，当真正的主要敌人是来自外部时，能够迅速团结周围和自己计划的人民群众是至关重要的。例如，在第二次世界大战期间，苏联当局称战争为人民和神圣的，土地为俄罗斯的。这是绝对必要的，敌人是外部的，对人民群众来说，确定斗争的确切方向至关重要：我们是俄罗斯人，他们是德国人。当时对群众意识的操纵是有意义的，符合苏联统治精英的目标：敌人客观上是外部的。

    然而，现在，在乌克兰，根据萨卡什维利本人的看法，主要敌人是内部敌人：腐败。在办公室里有印章和计算器的官员，而不是在路障上持枪和传单的分裂分子，是我们的主要敌人。在反对他们的运动中，我们期望您呼吁我们团结起来。

    那么，为什么要分散注意力于虚假的敌人，升起错误的旗帜，米哈伊尔？

    如果您说俄语，甚至不尝试学习乌克兰语并过渡到它，您只会强调自己对打击官员腐败的明确立场，而不论他们的民族和语言，避免对民粹主义的怀疑和指责。
