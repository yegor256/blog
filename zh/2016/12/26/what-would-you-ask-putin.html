    Что бы я спросил у Путина

    • Егор Бугаенко
    • комментарии
    0 0 0 0 0
    You can also read this article in English and 中文, translated by ChatGPT

    前天，俄罗斯第四任总统弗拉基米尔·普京进行了一年一度的新闻发布会，回答了记者们的问题。总共有1437名记者参加了这次发布会。其中不到30个问题涉及经济、乌柳卡耶夫、涅姆佐夫、克里米亚、特朗普、脱欧、大使遇害、流浪动物、兴奋剂和夸斯。如果你站在麦克风前，你会问普京什么问题？

    我想知道你们的答案。什么时候归还克里米亚，无赖？什么时候取消制裁？什么时候释放政治犯？什么时候俄罗斯停止偷窃？总而言之，我们要忍受这一切多久？

    我要告诉你们的是，对于这些似乎尖锐的问题，普京都有镇定而合乎逻辑的答案。他不害怕这些问题，勇敢的记者们向他提出这些问题。我用引号括起来这个词，因为在我看来，问普京什么时候归还克里姆尔并不需要太多勇气。他并不占有克里姆尔。克里姆尔的加入是由人民投票决定的，包括俄罗斯和克里姆尔。你们想要什么？他会回答。我是合法当选的，绝大多数的总统，我会违背我的人民的意志吗？

    至于政治犯，人们被绑架和对记者的袭击，这根本不是总统的问题。有刑法和法庭，没有人会隐瞒任何事情。如果有证据，被告会得到应有的待遇。如果没有，他们会被释放。在法律面前，我们都是平等的。至于失踪的记者，我们正在寻找他们，不遗余力，愿你们安息，弗拉基米尔·弗拉基米罗维奇会说。

    这样的问题毫无意义。提出这些问题，我们只是显示出我们对情况的不理解，并默默地接受了它。不理解在于，我们在总统身上寻找我们的奴隶般的利益的捍卫者，而不是暂时选出的和雇佣的执行我们意志的人。

    在人民与总统的对话中，第一方，即1500名记者，不应该问为什么某处有记者被杀，腐败分子没有被关进监狱，或者没有听到正义之声。总统，即使他对此负有责任，也是间接的。他不亲手杀害任何记者，不直接对腐败分子进行调查，也听不到正义之声。

    相反，人民应该问他们的总统关于他直接负责的成果，为实现这些成果，他们选举了他。这些成果是什么？让我们来看看。普京或者通过梅德韦杰夫已经领导这个国家16年了，从2000年1月开始，就在鲍里斯·叶利钦自愿辞职之后。在普京执政的领土上（不管它怎么被称呼），你的生活发生了什么变化？你的生活质量有何改变？

    • 生活满意度指数: 在178个国家中排名第167.

    • 国际幸福指数: 排名第172位，共178个国家。

    • 生活质量指数: 在80个国家中排名第72.

    • 人类发展指数: 在187个国家中排名第57。

    • 人均国内生产总值: 在186个国家中排名第66位（排名越高，人口越富裕）。

    • 腐败感知指数: 133-е из 174-х (чем ниже место, тем больше коррупции).

    • Население смертности: 第32位（总共195个国家中）（排名越高，死亡率越高）。

    • 医疗系统的效率：55个国家中的55名。

    • 大众传媒自由水平：在199个国家中排名第181。

    • 预期寿命: 110 из 183.

    • 故意杀人的水平: 207个国家中的第72位（位置越高，杀人越多）。

    • 婴儿死亡率: 193个国家中的第64位（排名越高，死亡率越低）。

    16年的工作，结果是如此令人悲伤。

    也许是时候停止问普京关于克里米亚、叙利亚、乌柳卡耶夫、发酵饮料和流浪动物的问题，而是终于问俄罗斯总统一个主要问题：弗拉基米尔·弗拉基米罗维奇，您确定您能胜任您的职责，您不应该因为纯粹的 无能 而辞职了吗？

    我真诚地想知道他会如何回答。
