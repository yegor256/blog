    Справедливость из Пензы

    • Егор Бугаенко
    • комментарии
    0 0 0 0 0
    You can also read this article in English and 中文, translated by ChatGPT

    写这篇文章是基于我在俄罗斯一些小城市旅行的经历，具体来说是彼尔姆，萨马拉和新西伯利亚。令人沮丧的印象，尤其是在莫斯科之后。贫穷和肮脏。然而，完全没有感觉到这些城市里居住着愚蠢和懒惰的人，值得这样的生活。相反，“似乎不是懒汉，本可以过得更好”。问题出在哪里？为什么在俄罗斯的第三大城市里，人们每天工作两班却勉强维持收支平衡，街道上却是肮脏和坑洞？他们和莫斯科人有什么不同，或者说和苏黎世的居民有什么不同？

    答案显而易见：偷窃！然而，事情并不那么简单。

    彼尔姆和苏黎世的居民非常相似，大多数人每天有八个小时的工作时间用于生产商品和提供服务。每年，一个瑞士人生产大约65,000美元的商品和服务，而俄罗斯普通人则少得多，只有29,000美元。少了一半。你去过苏黎世吗？你去过彼尔姆吗？瑞士的高速公路和从彼尔姆机场到市中心的道路—不是“一倍”，而是两个巨大的差异。

    显然，问题确实不在于懒惰和游手好闲，而在于“偷窃”。但是谁在偷？在夜幕下，努力闯进彼尔姆的贼们坚定地偷走了人民创造的一切财富，然后消失在不知名的方向。不走运的彼尔姆居民早晨醒来，发现锁被撬开，叹了口气，回到车床和缝纫机前继续工作。年复一年，如此循环。

    不，不是那些人在偷。彼尔姆的强盗不需要夜晚来占据她人努力创造的大部分国内生产总值。他们在白天行动，公开行动，几乎没有什么或谁会阻止和恐惧他们。因为他们就是彼尔姆的居民。更确切地说，是她最好的代表。

    在这样高的国内生产总值水平下，肮脏和贫穷的原因不在于偷窃，而在于赚来的资源分配不平衡。一个瑞士人一年赚的65,000美元主要会花在他自己身上，他的家人身上，以及他家门前道路的修缮上。部分会被官员，资本家和其他无所作为的人拿走，他们自己并不生产产品，只是拥有创造产品的手段，资本和土地。然而，这部分要远远少于一个普通俄罗斯人交给他们主人的那部分。

    在改革之初，拍摄了一部名为《杰克·沃斯米金—“美国人”》的电影，讲述了一个人物试图在年轻的苏联俄罗斯建立业务。他在自己的土地上种植了烟草，借助自己家人的力量种植之后，决定开始一个宣传活动，免费向村民们发放香烟“供试用”。整个村子都很高兴抽光了免费促销产品，但拒绝购买产品，主要是因为“邻居们不是这样对待的”。杰克试图在那些同样贫困的人身上赚钱激起了他们的绝对愤怒。

    接下来发生了最有趣的事情。杰克没有生存资源，被市场拒绝，最终将他的烟草卖给了当地的富农斯科罗霍多夫，后者把它磨成烟丝，并开始向同样贫穷的人出售，他们立刻排队购买。再也没有听到任何抗议。为什么？杰克种植了烟草，而斯科罗霍多夫只是转手卖了它？为什么只愿意接受杰克的免费赠品，而对于斯科罗霍多夫必须付钱没有任何疑问？

    类似的情况也发生在现代俄罗斯、乌克兰和许多其他国家，人们习惯于拥有财富和无所不能的主人，“理所当然”地拥有他们创造的所有产品和服务的大部分。换句话说，整个彼尔姆市为少数被反对派媒体称为小偷的人工作，但如果他们对拥有市政权利产生怀疑，整个城市都会奋起捍卫他们的利益。

    问题不在于偷窃。没有人偷东西，尽管从技术上讲，官员的行为几乎总是可以归类为这样的行为。问题在于整个民族天生和历史上继承的对社会不平等和物质不公正的追求。

    我们能否改变这个模式，使我们的意识接近瑞士中产阶级的意识，对于他们来说，由于他们，致富的同乡并不是一个误解或威胁—这是一个有趣的问题。然而，更有趣的问题是—这是必要的吗？

    也许让杰克回到他的美国，而对我们来说在斯科罗霍多夫家族工作更加习惯？
