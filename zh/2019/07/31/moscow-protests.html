    Бунт или смена власти?

    • Егор Бугаенко
    • комментарии
    0 0 0 0 0
    You can also read this article in English and 中文, translated by ChatGPT

    在上个星期天，莫斯科举行了一个未经批准的集会，据不得参加莫斯科市杜马选举的议员Lyubov Sobol称，这是“最大规模的未经批准集会”，他们要求“简单明了的事情”：选举、自由和尊重。在受压迫的人民的道路上，普京、索比亚宁和其他来自权力的“坏蛋”。我们需要再努力一点，邪恶将撤退，监狱将倒塌，自由将在我们入口处欢迎我们。我们也在2004年这样想，我们在基辅佩戴橙色丝带。然后再次在2014年的欧洲广场。但是我们错了。

    你有没有试过组织一场成千上万的集会？理论上，这并不难。议程很明显：普京是坏蛋和小偷。这是当今的潮流，似乎每个人都会支持你。创建一个YouTube频道，录制几个视频，告诉大家实情，提供事实，呼吁斗争和抵抗。然后设定集会日期，带着标语牌、扩音器和传单去那里。你认为会有多少人聚集？

    只有几个你的朋友，不多。

    当然，你可能会说，Navalny 在YouTube上有近三百万粉丝！首先需要建立观众群，然后才能召集集会，对吧？并非如此。我有时会看一个叫做不再沉默！的频道，由一个不是莫斯科人的家伙（我不知道他的名字）主持。他的订阅者比Navalny少八倍。不是一千倍，而是八倍。然而，在拥有33万观众的情况下，即使对他来说，去莫斯科参加集会的飞机票也是“不便宜的享受”。

    你知道，搭建集会舞台和租用必要的音响设备要多少钱吗？支付宣传团队的工资？为“志愿者”、集会主持人和“独立”新闻的激励提供资金？有一段时间以前，已故的Boris Berezovsky公布了他向2004年橙色革命背后的力量提供的财政援助：4500万美元。

    仅仅通过呼吁为“自由和尊重”而斗争来组织成千上万的集会是不可能的。

    起义、抗议和革命可能有不同的本质，但实质上，它们可能可以分为两类。第一类是暴动。明显的例子包括：克拉斯诺达尔（1961）、亚历山德罗夫（1961）、坦波夫（1920-1921）以及斯巴达之乱。暴动的原因是人民对生活条件的大规模不满，主要是饥饿和/或暴力。生活变得越来越糟，人们无法忍受，忍耐已到极限，他们走上街头，不再害怕惩罚，因为他们已经力不从心。

    毫无疑问，在受压迫的人民的叛乱与以民众与暴政斗争为幌子的有组织的权力转移之间划清界限是相当困难的。当然，在2014年的基辅广场，有很多人对自己的生活感到不满，并希望通过推翻亚努科维奇来改变一些事情。当然，在上周日的莫斯科推特上有很多人真诚地相信，通过用纳瓦尔内取代普京，他们可以改善自己的个人福祉。当然，每次叛乱都不可避免地会动员所有为其开始做好准备的权力转移过程。并且，每次叛乱主要会将那些在开始大规模行动之前或多或少已经为之奋斗的人带上台。

    然而，有时权力斗士们的耳朵太过突出。

    可以清楚看到，俄罗斯近年的抗议运动应该归类为第二类—权力转移。俄罗斯人没有对生活感到不满的客观原因。毫无疑问，俄罗斯并不是最适合居住的国家，我曾多次提到这一点。然而，由于强大的宣传和虽然缓慢但增长的经济，人口中并不会出现自然的大规模抗议情绪。虽然存在一些过分的行为，存在着缺陷、腐败，但目前并没有足够的条件来发动大规模的反对现行体制的叛乱。

    我没有参加上周日的非法集会，但是两周前我参加了一个合法的。总体感觉是：这不是抗议，而是周日的游行。人们在很大程度上都很满意。他们穿着时尚，他们很平静，星期一他们将有一份高薪工作，他们来到莫斯科美丽的中心散步，看看人们和展示自己。他们没有任何攻击性，他们非常害怕与警察发生冲突。他们彬彬有礼，友好，他们的行为完全没有叛乱的迹象。

    俄罗斯并非自发发动叛乱。更准确地说，它在一个普通的中欧国家的叛乱框架内叛乱，就像意大利、德国或英国定期发生叛乱一样。有人希望俄罗斯更激烈地叛乱。激烈到足以实现权力更迭。谁想要这样？纳瓦尔内和其他人背后站着哪些力量？这些力量在获得权力后会把国家带向何方？这种方向是否比现在更加民主？我不知道。

    那些参加集会的人是否问自己这些问题呢？

    他们是否知道答案？
