    245 лет сепаратизма, от Вашингтона до Захарченко

    • Егор Бугаенко
    • комментарии
    0 0 0 0 0

    1772年6月，在今天的罗德岛州纳拉甘塞特湾（美国）一小群走私者袭击了一艘政府军舰，打伤了舰长，解除了船员的武装，抢劫了船只并将其焚毁。这标志着美国殖民地为摆脱英国统治而发动的独立战争的开始，这场战争的英雄是后来成为首位总统的乔治·华盛顿上校。将近两个半世纪后的2014年3月，几千人攻占了顿涅茨克行政大楼并在其上升起了俄罗斯国旗，这标志着顿巴斯战争的开始，现在我们称之为与俄罗斯的战争（根据波罗申科的说法）或独立斗争（根据扎哈尔琴科的说法）。这两个事件之间有什么共同之处？

    在地球上曾经发生、正在发生和将要发生许多战争和军事冲突。它们在规模（例如第二次世界大战有62个参与国家和6000多万人死亡，相比之下，持续38分钟的英国-桑给巴尔战争造成了大约570人死亡）、性质（例如，战争可能是解放性的、内战、宗教性的，或者根本是暴动、起义或革命性质的）、持续时间、紧张程度以及其他方面有所不同。然而，根据哥伦比亚大学政治学和经济学教授马西莫·莫雷利的观点，大多数冲突都源于“未履行承诺”的问题。

    如果谈到为独立而进行的战争，那么这种未履行的承诺，显然是关于不要带着自己的规则“闯入别人的园地”的无形但显而易见的承诺。因此，数百年前的高卢、马其顿或意大利的罗马省起义与18世纪印第安人与美国人的斗争或20世纪车臣人与俄罗斯联邦的冲突在本质上并没有太大区别。在所有情况下，园主们对外来的“规则”感到不满。

    显然，这是人类的本性：我们不喜欢被外部控制，即使这是出于好意。我们愿意忍受，直到明显的利益存在（例如美国的州），或者直到我们被武力制止（例如苏联的共和国）。

    支持关于在大多数情况下，一旦走上分裂之路，领土不可避免地脱离“压迫者”控制的论点的简单事实是，在1900年，地球上只有少于50个国家。例如，印度、加拿大、波兰、芬兰、乌克兰或澳大利亚在20世纪初都不存在，它们都是其他国家的一部分或殖民地。100多年过去了，目前联合国有193个个国家。

    显然，要将国家数量增加四倍，需要走过许多分裂行动的道路，无论是否涉及军事行动。

    因此，不管我们如何看待它，分裂主义可以被称为新世纪改变世界地图的稳定趋势。没有客观上的消退理由。最近在英国和加利福尼亚发生的事件是明显的证据。

    现在让我们问一个问题：顿巴斯最终会属于哪个领土组？乌克兰是否会通过武力手段将其保留在其管辖范围内，并且等待像车臣或巴勒斯坦那样的命运，或者会形成类似科索沃的新国家，边界将被确认，并且武器将被放下？
