    Совесть в балаклавах

    • Егор Бугаенко
    • комментарии
    0 0 0 0 0

    在美国第45任总统唐纳德·特朗普的就职典礼期间，抗议者们走上了华盛顿的街头，其中217人被拘留。他们手持标语牌，最终开始推翻垃圾箱，砸碎橱窗，和点燃汽车。许多人穿着黑色服装，有一定程度的组织行动。不管我们对特朗普及其政策持何种态度，爆炸和火灾、戴面具的人和警车警报声仍让我们恐惧。我们依靠警察的可靠性。然而，这现象也有其反面。

    自斯巴达克时代以来，反对当权者的起义在这个星球上经常发生。目标始终只有一个 — 自由。动机总是不同的。例如，经济动机 — 税收上涨，引发了拉辛起义的发生（尽管有疑虑）；政治动机，如罗马尼亚革命 — 共产主义政权被亲西方的民主取代；种族和民族动机，如纳戈尔诺-卡拉巴赫或科索沃；还有纯粹的武力动机，如十月革命 — 革命是布尔什维克党所必需的，他们实现了这一点。

    根据LATININA的精辟说法，“世界上的革命远远多于民主”。确实，大多数起义并没有改善那些为之起义的人们的生活。虽然新人掌权，但秩序仍然基本上保持不变，或几乎保持不变。最近的乌克兰革命在马丹广场，如橙色革命和尊严，是一个鲜明的例证。

    这并不是叛乱分子、革命者和手持摩洛托夫鸡尾酒的恐怖分子的错。甚至不是领导和激励他们的罗伯斯庇尔。错在我们身上，那个唯一通过改变旗帜和宣言自由的声明而无法改变的绝大多数。回想一下，兰斯洛特试图在马克·扎哈罗夫的著名电影中杀死龙，但龙不断重生，因为这是“城市最好的人”所要求的。

    因此，革命，起义，叛乱，燃烧的轮胎，屋顶上的狙击手，酒醉的学生们在钢筋上狂欢，破碎的橱窗和烧毁的公共汽车 - 这一切都毫无意义，毫无影响？显然，聪明人应该尊重权力，呆在家里，看连续剧，不要走上玛丹头顶上托着锅，通过民主途径表达自己的政治诉求，每隔几年就投票放入选票箱中？

    是的，也许。然而，请想象一下，如果没有普加乔娃、丹东、穆拉维耶夫、巴枯宁、涅恰耶夫、托洛茨基、卡斯特罗、杜达耶夫和许多其他人，他们反抗权力，首先冒着生命危险。一些人被权力打败，一些人被消灭，一些人成功战胜并成为新的权力，以便与新的革命者和革命作斗争。重要的不是每次起义的个人实际成就，尽管这些成就很多，没有它们我们就没有养老金，没有休息日，没有带薪假期，没有言论自由，没有人权，没有平等，没有司法。重要的是别的。

    重要的是社会中始终存在着对权力的不顺从精神和准备起义的态度。正如让·若尔斯所说，“只有有良知的地方才可能发生革命”。这些半醉的年轻人戴着头巾，打破我们喜爱的餐厅橱窗，在我们家的墙上涂鸦，正是由我们和您的良知组成的。没有他们，我们将成为奴隶。

    维克多·雨果认为“革命 - 是从虚伪走向真实的回归”，显然，他将“真实”理解为我们的能力和愿望成为自由的。

    不要急于谴责造成破碎橱窗和翻倒垃圾箱的起义者。也许几百年后，您的后代会为他们的自由而感激他们。
