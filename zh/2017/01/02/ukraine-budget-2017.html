    Честный бюджет от продажных депутатов

    • Егор Бугаенко
    • комментарии
    0 0 0 0 0
    You can also read this article in English and 中文, translated by ChatGPT

    上周，乌克兰最高拉达接受了政府提出的2017年预算。几天后，在NewsOne的节目中，穆斯塔法·纳伊姆分析了这一事件，并赞扬了政府的及时性，但称最后阶段是一场闹剧，因为在他看来，反对派收受了贿赂以支持政府。相反，我认为纳伊姆的发言才是闹剧。

    我想到了这样的比喻。想象一下，我们正在为体操比赛培训一名运动员。一切都不错，但他经常从高低杠上摔下来。除了其他问题外，他还患有脑瘤。

    我们知道这种疾病，但他还有其他困难。首先，我们怀疑他穿的紧身运动衫太紧，影响了动作：我们尝试选择更合适的服装。其次，他的前臂肌肉似乎很弱：显然，应该增加更多针对这一肌肉群的练习。第三，很明显，他体重超标：减掉几公斤可能会有所帮助。

    所以，我们有四个问题：紧身衣、弱肌肉、超重和脑瘤。

    当然，我们明白，他很可能经常在垫子上摔倒是因为失去意识，而不是因为应该让他继续练习高低杠，而是应该接受低生存率的强化化疗。我们知道这一点，但还有其他问题，比如紧身衣和超重。这些问题也需要解决，不是吗？

    听起来荒谬，对吧？如果运动员的生命本身受到威胁，那么超重和紧身衣又有什么关系。不解决主要问题，甚至提及其他问题都毫无意义。

    现在乌克兰正在发生同样的事情。这个国家的整个政府机构都被腐败腐蚀。乌克兰在一个著名排名中排名第130位，只有二十多个国家比我们更糟糕，其中包括刚果、乌干达、叙利亚、伊拉克和索马里。腐败就像一种恶性肿瘤，让我们的国家无法站起来。

    预算又如何呢？它被接受的时间或者延迟有什么区别？它是否是高质量的，或者有错误？这份文件在一个没有任何权力机构正常运作的国家中到底有什么作用。

    穆斯塔法，您建议我们为一名患有脑瘤的运动员减肥？您是不是不理解情况，或者您也被支付，就像您指责的那些反对派一样。
