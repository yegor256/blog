    Саботажника Саакашвили геть!

    • Егор Бугаенко
    • комментарии
    0 0 0 0 0
    You can also read this article in English and 中文, translated by ChatGPT

    前格鲁吉亚总统米哈伊尔·萨卡什维利在2015年2月成为波罗申科的顾问。三个月后，他获得了乌克兰公民身份和奥德萨州长的职位，月薪为280美元。半年后，他被剥夺了格鲁吉亚公民身份。在接下来的一年里，萨卡什维利对腐败提出了一系列猛烈指控，包括针对阿瓦科夫、亚采纽克以及波罗申科本人。2016年11月，他宣布辞去奥德萨职务，波罗申科解雇了他，一举摆脱他以及顾问。昨天，萨卡什维利被剥夺乌克兰公民身份。

    这是什么？我们在接下来的两年半里究竟成为了什么见证者？这位有经验的格鲁吉亚政治家和革命家，过去几年格鲁吉亚取得的独特成就都与他有关，竟然成为了一个无所作为的外行人，只会空谈而不会实干？

    还是这是两位老朋友之间的个人冲突，演变为政治对抗，导致其中一位更弱、不善于说乌克兰语的人失败？

    我认为，可以毫不犹豫地将过去两年萨卡什维利在乌克兰的悲剧视为一次失败的破坏行为。一开始，尼古拉斯维利成为该体系的共犯之一，显然是相信了其领导者的善意动机。然后，意识到自己的错误后，他试图从内部破坏这个体系，从乌克兰规模不大的地方职位——地方行政长官。但他并未成功。这个体系为了保持自身的稳定性，镇压了破坏行为，并将破坏者驱逐出境，而且很可能在不久的将来会试图惩罚他。

    与波罗申科的冲突并非如萨卡什维利在他的”告别“声明中所描述的那样。没有人与他结仇。这不是个人问题。只是商业—腐败，这是过去二十五年来整个国家建立和维持的基础，正在为自己的生存而战。

    我们是这个体系的一部分，我们是它的建筑材料，它的追随者，它的建造者和忠实的信徒。我们剥夺了萨卡什维利的公民权，没有让他说出他所知道的有关阿瓦科夫等人的腐败的一切，并且在他最后几个月可能出现的那些电视频道上没有仔细听完他。

    他妨碍了我们。他试图破坏我们的腐败稳定。他破坏了我们的规则和工作原则。我们把他赶走了。他活该。

    消灭破坏者！乌克兰万岁！
