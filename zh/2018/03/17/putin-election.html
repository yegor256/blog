    Что сделал Путин, чтобы завтра победить

    • Егор Бугаенко
    • комментарии
    0 0 0 0 0
    You can also read this article in English and 中文, translated by ChatGPT

    明天在俄罗斯将举行总统选举，弗拉基米尔·普京 将获胜。这将是他的第四个总统任期。事实上，他已经掌权18年，自2000年1月接管权力以来。几乎1.5亿人口的世界最大国家将再次授权这个人管理他们的命运。很难不承认，这是值得尊重的结果。他是如何做到的呢？

    普京政策成功的主要因素是对他所处理的人的冷静和适度的态度。当菲赫特万格在与斯大林交谈时对他过分且无味的崇拜提出批评时，斯大林根据他的说法，”为他的农民和工人道歉，因为他们忙于其他事情，无法培养出好品味来”。看来，普京也具有类似的冷峻的讽刺态度，这体现在他对人民的统治风格中。

    以下是带来这些结果的主要方法：

    独裁。俄罗斯就是普京，普京就是俄罗斯。很难说幕后决策是如何做出的，谁的意见有多大权重，他们之间的权限是如何真正分配的。对于普通公民来说，他们的感觉是这个国家由一个人控制。人们对这个人有何看法，无论是爱或恨，都不是很重要。重要的是看法是集中的。这有助于维持局势的控制。

    崇拜。在执政六年内打造一个统治者个人崇拜的形象很困难，但在近二十年内做到这一点是可能且必要的。崇拜的存在使得统治更容易，因为这提供了依靠的权威。普通公民通常不倾向于深入思考和批判性思维。复杂的逻辑结构对他们而言是困难且令人疲惫的。他们需要清晰的答案来解决当前问题。逻辑结尾“因为这是普京”可以轻松而自信地回答大多数问题。

    开放。每年与记者举办的新闻发布会，与学生举办的公开课，参观工厂，接受外国记者的采访，所有这些都创造了一个开放且可接近的形象。他说什么并不是很重要，他说得不错，重要的是看到他与人们开放交流。这营造出“熟悉”和“亲近”的感觉。在街上遇到知名演员，我们微笑并点头示意，却没有得到任何回应，这让我们感到惊讶和沮丧，因为我们习惯将这个人视为亲密朋友。从政治人物公开性的角度看，同样的效果也体现在普京身上，他早已成为数百万俄罗斯人的亲人。

    侵略性。1999年还未成为总统时，普京承诺“在厕所里消灭恐怖分子”，尽管措辞粗鲁，但对普通选民来说听起来很好。在这样的表述中有一个明确的公式：我们反对他们。“他们”是谁，我们将“消灭”他们，这并不重要。重要的是，有一个“我们”，他们不会让自己受辱。普京执政这些年来，俄罗斯的敌人很多，如果不够多，他们就创造出来，引爆房屋。团结数百万人围绕和平的想法是不可能的。团结需要仇恨和战争。无论是车臣，格鲁吉亚，普林德斯特罗夫，叙利亚，顿巴斯还是整个北约。人们想知道敌人在哪里。领导者的任务是找到或创造这个敌人。普京成功地做到了。

    力量。甚至马基雅维利认为，只有通过恐惧才能控制人群。爱情、尊重、理智，甚至金钱都无法阻止人群在失控时。通过将强制结构（特别是联邦安全局）几乎无限的权力授予，普京简化并凝聚了权力的形象给他的选民 — 大家都知道要害谁，谁掌握权力，它来自哪里，以及要向谁屈服。考虑到普京所面对的人群，这对国家的负面影响要少于正面影响。

    无情。定期逮捕和残酷惩罚自己最亲近的人创造了一个公正统治者的形象。重要的不是某个乌柳卡耶夫到底犯了什么罪，重要的是惩罚是多么严厉和不可避免。人民害怕侍卫，而侍卫害怕国王 — 这个公式已经有几百年了，而且行之有效。人民相信国王对于违法和腐败一无所知。人民希望看到国王惩罚那些嚣张的杀人犯。人民给国王写信，希望他公正。这种与权力交流的方式对人民来说是方便和清晰的。

    虚伪。以久经不衰的日里诺夫斯基为首的剧场反对派清楚地表明，普京擅长玩双重游戏并与对手达成协议。他并不像许多独裁者那样“把人扔进厕所”。他付钱给某些人，比如卡德罗夫，与某些人分享舞台和荣耀，比如与索布恰克和纳瓦尔尼，还杀死某些人 — 在每种情况下，都做出了个别决定。尊严、良心和人类尊严完全被忽视，正如一位大批评人物应该做的那样。人们并不在乎领导者是怎样的，他们关心的是自己的舒适。领导一个关心人民舒适而不是自己良心纯洁的人将会受到尊重并被重新选举。

    自信。不承认自己的错误是普通人的一个令人反感的品质。然而，在一位伟大领导者身上，它变成了一种美德。人民无法理解错误的原因和所得出的结论。错误就是错误。按照马基雅维利的说法，一个犯错的领导者，还承认错误的，是一个软弱的领导者。普京从未批评自己，也从来不承认错误，同时继续许诺却不兑现，甚至撒谎。从纯粹的人类角度来看，这让我们感到愤慨，但在百万人的规模下，这是一个有效领导者的品质。

    爱国主义。智慧、目标、活力和决心是领导者的重要品质。然而，爱国主义更为重要。人民可能偶尔原谅错误，但绝不会原谅不忠。与其他国家和领导人持续紧张甚至敌对的关系明确向普京的公民发出了信息：“我与你们一起，而不是与他们一起”。国际制裁和俄罗斯在世界舞台上的孤立在实际意义上很少影响普通公民，但通过媒体的报道生动地展示了总统的爱国主义。

    结论是 — 考虑到他所处的环境和选择他的人，普京做得都是正确的。在与其他人、在其他国家、在其他条件下，显然可以做出不同的选择。

    明天选举，你会投票支持普京吗？
